RipeSight is a Computer Vision AI built to identify fruit and vegetables, determine if they are ripe or rotten, and even allow the user to view, share, and create recipes based on a given. RipeSight uses an object detection and computer vision API known as YOLOv5 in synergy with an object labeling model to quickly and accurately identify fruit and vegetables. Our model also makes use of free recipes from All Recipes to provide a large database of recipes for the user to choose from.
